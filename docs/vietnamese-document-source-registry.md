# Vietnamese Document Source Registry

This registry records safe source categories for Vietnamese OCR testing and
synthetic sample design. It is a documentation-only allowlist and risk guide.
Do not download, commit, or vendor any external documents, datasets, receipt
images, invoice images, or customer files from these sources.

## Project Safety Policy

### Public Repo Allowed

- Synthetic samples generated by this project.
- Blank or demo structures recreated with fake data.
- Source registry docs.
- Generator code.

### Public Repo Not Allowed

- Downloaded real invoices.
- Customer documents.
- MC-OCR raw images.
- Kaggle dataset files.
- QR codes from real invoice or payment documents.
- Bank account information.
- CCCD, CMND, passports, or faces.
- Real tax codes, phone numbers, emails, or addresses.
- Documents containing signatures or stamps that identify real entities.

### Local-Only Allowed With Caution

- Anonymized real documents under ignored paths.
- Official MC-OCR dataset files only after checking and accepting the terms.
- Downloaded blank templates for private layout analysis.

Local-only files must stay under ignored paths and must not be referenced by
committed fixtures, tests, or docs in a way that exposes private details.

## Recommended Next Steps

1. Do not download or commit external documents.
2. Use the registry to design synthetic template generators.
3. Start with 5 synthetic generator types:
   - retail invoice
   - VAT invoice
   - payment receipt
   - delivery note
   - warehouse in/out slip
4. Store local-only external docs only under ignored path:

   ```text
   storage/dev/private_samples/
   ```

5. Commit only synthetic outputs that pass a privacy scan.
